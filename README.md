# 💼 Personal Portfolio

Welcome to my personal portfolio repository! This project showcases my work, skills, and professional experience as a developer. Built with modern web technologies, it serves as both a digital resume and a central hub for my projects and blog posts.

## 🚀 Live Demo

Visit the live portfolio: [portfolio-esrafil.netlify.app](https://portfolio-esrafil.netlify.app/)

## ✨ Features

- **Modern Tech Stack**: Built with Next.js, Tailwind CSS, and TypeScript for a fast, responsive, and type-safe experience.
- **Dynamic Content**: Projects and blog posts are managed via MDX files, making it easy to update and extend.
- **Contact Form**: Integrated email functionality for direct communication.
- **Clean & Minimalist Design**: Focused on readability and a professional user experience.
- **Performance Optimized**: Leveraging Next.js for optimal loading speeds and SEO.

## 🛠️ Built With

- **[Next.js](https://nextjs.org/)** - React framework for production
- **[TypeScript](https://www.typescriptlang.org/)** - Static type checking
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework
- **[MDX](https://mdxjs.com/)** - Markdown with JSX components for content
- **[Zod](https://zod.dev/)** - TypeScript-first schema validation
- **[Sonner](https://sonner.emilkowal.ski/)** - Toast notifications
