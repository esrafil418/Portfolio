## 📖 About The Game

Memory Card Game is a classic matching game where you flip cards to find matching pairs. The game tracks your **score** and **moves**, and celebrates when you complete all matches!

### 🎯 How to Play

1. Click on any card to flip it
2. Try to find another card with the same emoji
3. If they match, the cards stay flipped!
4. If they don't match, they flip back
5. Match all 8 pairs to win!
6. Try to complete the game with the fewest moves possible

## ✨ Features

- ✅ **8 pairs of emoji cards** (16 cards total)
- ✅ **Score tracking** - earn points for each match
- ✅ **Move counter** - track how many moves you make
- ✅ **Flip animation** - smooth 3D card flipping
- ✅ **Match detection** - automatically checks for matches
- ✅ **Win message** - celebrate when you complete the game
- ✅ **Reset button** - start a new game anytime
- ✅ **Responsive design** - works on phones, tablets, and desktops
- ✅ **TypeScript** - type-safe code for better reliability
